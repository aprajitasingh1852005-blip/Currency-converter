# Currency Converter 

A simple and responsive Currency Converter web application that allows users to convert an amount from one currency to another using real-time exchange rates from a public API.

## Features
- Enter amount to convert
- Select source currency
- Select target currency
- Fetch real-time exchange rates
- Display converted result clearly
- Loading state while fetching data
- Error handling for network issues

## Technologies Used
- HTML
- CSS
- JavaScript
- Exchange Rate API
